# AlgrithmClass
包含“并查集”，“最短路”等算法的模板类，可以应付LeetCode的板子题目，也可以作为算法学习的参考代码。

## 现包含以下内容：
- 1 UnionFind 并查集类

``` C++
UnionFind(int n) 

```

- 2 BOM_UnionFind 基于std::unordered_map的并查集

``` C++
BOM_UnoinFind()

```

- 3 MonotonicSatck 单调栈

``` C++
MonotonicStack(Vector<T>, cmp, isleft=false)

```

- 4 PSO 粒子群算法

``` C++
PSO(typename func f, int vnum, double bound)

```
